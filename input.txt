Python 基礎
　if文
　　単純な条件分岐
　　　数値の例
　　　　if文は、条件によって処理を帰ることができる
　　　　code x = 7\nif x == 7:\n    print('x は7です')\nelse:\n    print('xは7ではない')

　コレクション
　　リスト

　　　基本的なリスト内包表記
　　　　りスト内包表には、元のリストを利用して新しく作る場合や、フィルターに掛けて必要な要素だけを取り出す時に便利である
　　　　code A = [3, 2, 6]\nB = [a * 2 for a in A]\nC = [a - 3 for a in A]\nprint('A = ', A)\nprint('B = ', B)\nprint('C = ', C)
　　　　code # 出力\nA =  [3, 2, 6]\nB =  [6, 4, 12]\nC =  [0, -1, 3]

　　　リスト内包表記を使ったフィルター的な
　　　　リスト内包表記で、ある条件を設定して、その条件に当てはまるものだけを要素とする、ということができる
　　　　code A = [3, 8, 5, 2, -8, 4]\nB = [a for a in A if a >= 3]\nprint('B = ', B)
　　　　code # 出力\nB = [3, 8, 5, 4]

　　タプル
　　　基本的な使い方
　　　　タプルは複数の要素を持ちたい時にまとめて持つのが便利
　　　　リストとの違いは、リストは要素を変更したり追加したりできるが、タプルは変更不可な点などである
　　　これって実は...
　　　　値のスワップ時や複数代入時に、
　　　　code a, b = 3, 9\nprint(f'a = {a}  b = {b}')\na, b = b, a\nprint(f'a = {a}  b = {b}')
　　　　code # 出力\na = 3  b = 9\na = 9  b = 3
　　　　といったコードを書くことは無いだろうか。これはタプルのパック・アンパックという機能を使っているものである
