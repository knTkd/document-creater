2. コレクション

2-1. リスト

2-1-1. 基本的なリスト内包表記

りスト内包表には、元のリストを利用して新しく作る場合や、フィルターに掛けて必要な要素だけを取り出す時に便利である 
A = [3, 2, 6]
B = [a * 2 for a in A]
C = [a - 3 for a in A]
print('A = ', A)
print('B = ', B)
print('C = ', C)
# 出力
A =  [3, 2, 6]
B =  [6, 4, 12]
C =  [0, -1, 3]

2-1-2. リスト内包表記を使ったフィルター的な

リスト内包表記で、ある条件を設定して、その条件に当てはまるものだけを要素とする、ということができる 
A = [3, 8, 5, 2, -8, 4]
B = [a for a in A if a >= 3]
print('B = ', B)
# 出力
B = [3, 8, 5, 4]

2-2. タプル

2-2-1. 基本的な使い方

タプルは複数の要素を持ちたい時にまとめて持つのが便利 リストとの違いは、リストは要素を変更したり追加したりできるが、タプルは変更不可な点などである

2-2-2. これって実は...

値のスワップ時や複数代入時に、 
a, b = 3, 9
print(f'a = {a}  b = {b}')
a, b = b, a
print(f'a = {a}  b = {b}')
# 出力
a = 3  b = 9
a = 9  b = 3
